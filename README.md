# 📊 Sales Analyzer Python Mini Project

A simple Python-based sales analysis project that reads sales data from a CSV file, processes it, calculates product and category-wise revenue, and exports filtered data into separate CSV files.

---

## 🚀 Features

- Read and clean sales data from CSV files
- Calculate total revenue for each product
- Analyze category-wise sales performance
- Filter records by category
- Export filtered category data into new CSV files
- Practice Python file handling and data processing

---

## 🛠️ Technologies Used

- Python
- CSV Module
- Jupyter Notebook

---

## 📈 Project Workflow

1. Load sales data from CSV file  
2. Clean and format the data  
3. Calculate product-wise revenue  
4. Analyze category-wise sales  
5. Filter data by category  
6. Export category data into separate CSV files
